# K Nearest Neighbors Project

## Project Overview  
In this project, I used the K-Nearest Neighbors (KNN) algorithm to classify data based on different features. The goal was to build a model, evaluate its performance, and find the best value of K.

## Project Steps  
- Visualized the data using a pairplot  
- Standardized the features using StandardScaler  
- Split the data into training and testing sets  
- Trained a KNN model  
- Evaluated the model using confusion matrix and classification report  
- Used the elbow method to find the best K value  
- Retrained the model with the optimal K  

## Tools Used  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

## Conclusion  
The model was able to classify the data with reasonable accuracy. By choosing a better K value, the performance improved and the error rate decreased.
